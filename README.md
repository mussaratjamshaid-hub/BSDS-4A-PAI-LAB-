# AI Sentiment Analyzer

## Project Overview
This is an AI-based Sentiment Analyzer web application built using Flask and NLP. It detects whether text is Positive, Negative, or Neutral.



## Features
- Sentiment analysis (Positive / Negative / Neutral)
- Flask web application
- Simple UI
- Fast AI response
- Optional OpenCV support



## Tech Stack
- Python (Flask)
- NLP (TextBlob / NLTK)
- HTML, CSS, JavaScript
- OpenCV (optional)

## How It Works

User enters text → Flask backend receives it → NLP analyzes sentiment → Result is shown on UI.
