### AI Sentiment Analysis System
(Flask + NLP + Face Detection Web Application)


### 1. Introduction
The AI Sentiment Analysis System is a web-based application that analyzes human emotions using both text and facial expressions. It combines Natural Language Processing (NLP) and Computer Vision techniques to detect sentiment and emotions.
This project demonstrates how artificial intelligence can be implemented in a real-world web application using Python Flask.

### 2. Objectives
•	To analyze user sentiment from text input 

•	To detect emotions from facial images 

•	To develop a web application using Flask 

•	To apply NLP and OpenCV in a practical project 

•	To create a modern and user-friendly interface 

### 3. Technologies Used
Backend:

•	Python 

•	Flask 

Frontend:

•	HTML 

•	CSS 

### AI / ML:
•	Natural Language Processing (NLP) 

•	OpenCV (Face Detection) 


### 4. System Working
1.	The user provides input (text or image)
   
3.	For text:

o	The system analyzes the text using NLP 
o	It classifies sentiment as Positive, Negative, or Neutral

4.	For images: 
o	The system detects the face using OpenCV 
o	It analyzes facial expressions to predict emotion

### 6.	The result is displayed on the web interface 

### 7. Features
•	Text Sentiment Analysis 

•	Facial Emotion Detection 

•	Image Upload Functionality 

•	Responsive Web Design 

•	Flask-Based Backend System 

### 8. Project Structure
•	app.py (Main application file)

### Applications

### 9. Limitations
•	Accuracy depends on model quality 

•	Limited detection of complex emotions

•	Performance may vary with image quality 

### 10. Future Improvements
